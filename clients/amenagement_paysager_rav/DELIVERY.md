## Une seule étape : coller le snippet

Demandez à votre développeur web (ou faites-le vous-même dans le CMS) de coller le code ci-dessous **juste avant la balise `</body>`** sur toutes les pages où vous voulez afficher le chatbot.

```html
<script src="https://web-production-42cf20.up.railway.app/clients/amenagement_paysager_rav/chatbot/snippet.js"></script>
```

C'est tout. Le chatbot apparaîtra automatiquement en bas à droite de votre site.
